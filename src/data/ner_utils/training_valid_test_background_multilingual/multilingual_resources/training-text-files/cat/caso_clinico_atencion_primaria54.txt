3-Com es transmeten les ITS?
Via sexual. Per sexe vaginal, anal, bucogenital, sense preservatiu amb una persona infectada.
Via sanguínia. A través d'elements punxants o tallants compartits amb altres persones : fulles, agulles xeringues, útils de pírcing i tatuatge.
Via perinatal. Durant l'embaràs, el part o la lactància.

4- Com no es transmet el VIH?
Per viure o treballar amb una persona infectada.
Per donar la mà, besar, acariciar o abraçar.
Per compartir roba, plats, coberts, gots, dutxa o WC.
